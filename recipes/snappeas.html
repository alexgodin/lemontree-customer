Ingredients

Equipment

Pinch each snap pea at the top and peel away the string

Peel off chicken skin, Cut chicken into rough chunks and discard the bone.

Toss chicken with salsa to cover. Set aside.

Mince garlic, set aside.

Slice onions, set aside

Heat a pan with oil over high heat. Once pan is hot, add garlic.

Once garlic starts to smell, add chicken to pan. After 1 minute, add onion

Once onion has started to soften, add string beans.

The dish is done once string beans and onions start to brown. Optional: If you like your food spicy, add more salsa towards the end.
